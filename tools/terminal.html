Как настроить zsh ​

  1. устанавливаем https://iterm2.com/
  2. устанавливаем oh-my-zsh
sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
  1. устанавливаем тему:
sh
git clone https://github.com/romkatv/powerlevel10k.git $ZSH_CUSTOM/themes/powerlevel10k
  1. редактируем ~/.zshrc
sh
ZSH_THEME="powerlevel10k/powerlevel10k"
  1. перезапускаем терминал
  2. устанавливаем настройки
  3. iterm2 -> preferences -> profiles -> keys -> presets -> "Natural Text Editing" (чтобы можно было быстро перемещаться в строке терминала)
  4. Открываем стандартный терминал, настройки -> профили -> текст -> меняем шрифт на MesloLGS NF (чтобы иконки корректно отображались)
  5. Открываем VSCode, command + P, settings.json -> Open Settings Json -> вставляем строку "terminal.integrated.fontFamily": "'MesloLGS NF'", (тоже для иконок в терминале VSCode)
